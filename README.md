# arr-installer

A project for automating array installation and management.

## Overview

This project provides tools and utilities for installing and configuring arrays in your environment.

## Features

- Automated installation process
- Configuration management
- Easy-to-use CLI interface

## Installation

```bash
git clone https://github.com/yourusername/arr-installer.git
cd arr-installer
```

## Usage

```bash
uv sync
uv run main.py
```

## Requirements

- Docker
- uv
    - Python 3.8+
        - requests
        - tzdata
        - rich
        - questionary
- Git
